# Enhancing-cyber-security-with-AI-Intrusion-detection-using-CICIDS2017-dataset-
An AI-driven intrusion detection system using the CICIDS2017 dataset, applying feature engineering and classification models to detect and classify malicious network traffic.

🚀 Tech Stack
Python
Scikit-learn
Pandas
NumPy
Matplotlib

📂 Dataset
CICIDS2017 dataset
Contains labeled network traffic with multiple attack types and benign traffic
Includes flow-based features like packet count, duration, and byte statistics

⚙️ Key Features
✔ Data preprocessing and cleaning of large-scale network traffic data
✔ Feature engineering and selection for improved performance
✔ Implementation of classification models (Random Forest, Logistic Regression)
✔ Model evaluation using precision, recall, F1-score, and confusion matrix

📊 Results
Successfully classified benign and malicious traffic
Achieved reliable performance using standard evaluation metrics

🔄 Workflow
Data Loading and Exploration
Data Preprocessing and Feature Scaling
Model Training (Random Forest, Logistic Regression)
Performance Evaluation

▶️ How to Run
Clone the repository
Install dependencies using pip
Run the Jupyter notebook
